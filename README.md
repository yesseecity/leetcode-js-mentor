# LeetCode JavaScript 解決方案

## 專案簡介
此專案旨在作為您的 LeetCode 導師，提供一系列使用 JavaScript 實現的演算法問題解決方案。我們致力於提供清晰的解題思路、詳細的步驟引導，並附上符合最佳實踐的程式碼範例及解釋。每個解決方案都包含相關的測試案例，以確保其正確性。

## 專案特色
*   **清晰易懂：** 提供清晰、簡潔且易於理解的解釋和程式碼。
*   **逐步引導：** 鼓勵使用者獨立思考，透過逐步引導的方式幫助他們找到解決方案。
*   **最佳實踐：** 遵循 JavaScript 的最佳實踐和慣例。
*   **錯誤處理：** 協助使用者理解和調試程式碼中的錯誤。
*   **測試案例：** 提供相關的測試案例以驗證解決方案的正確性。

## 目錄結構
本專案的解決方案依據 LeetCode 問題的類型進行分類，主要目錄包括：
*   `array/`：陣列相關問題的解決方案。
*   `linked-list/`：鏈結串列相關問題的解決方案。
*   `string/`：字串相關問題的解決方案。
... (未來將根據需要新增更多分類)

每個問題的解決方案都將存放在對應的目錄中，例如 `array/two-sum.js`。

## 如何使用
1.  **瀏覽解決方案：** 您可以透過瀏覽專案目錄來查找特定問題的解決方案。
2.  **理解思路：** 每個解決方案檔案通常會包含註解，解釋解題思路和演算法步驟。
3.  **運行測試：** 您可以自行運行提供的測試案例，以驗證解決方案的正確性。

## Cline 的使用
作為 LeetCode 導師，Cline 可以協助您：
*   **獲取解題思路：** 向 Cline 提問特定 LeetCode 問題的解題思路。
*   **程式碼解釋：** 請求 Cline 解釋程式碼範例的細節。
*   **錯誤調試：** 當您的程式碼遇到問題時，Cline 可以協助您分析和調試錯誤。
*   **最佳實踐建議：** 詢問關於 JavaScript 演算法實現的最佳實踐。

**範例指令：**
*   「請解釋 `array/two-sum.js` 中的解題思路。」
*   「我的鏈結串列反轉程式碼有問題，可以幫我看看嗎？」
*   「如何為 `string/reverse-string.js` 撰寫更多測試案例？」
